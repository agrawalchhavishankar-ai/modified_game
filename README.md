A simple terminal-based car dodge game developed in C language using basic concepts like loops, arrays, functions and keyboard input.

In this game, the player controls a car and must dodge the falling obstacles. The speed of the game increases with time, making it more challenging. The player has a limited number of lives and can restart the game when it is over.

---

## FEATURES

* 3 Lives system
* Score system (score increases when obstacle is avoided)
* Difficulty increases automatically (speed increases with time)
* Restart system (Press 'R' to restart anytime)
* Obstacle appears randomly in 3 lanes
* Simple keyboard controls

---

## CONTROLS

LEFT Arrow  - Move car to left lane
RIGHT Arrow - Move car to right lane
R           - Restart the game
ANY KEY     - Exit on Game Over screen

---

## REQUIREMENTS

This program uses:

* <stdio.h>
* <conio.h>
* <windows.h>
* <stdlib.h>
* <time.h>

## GAME RULES

* Player starts in the middle lane.
* Obstacles fall from the top in random lanes.
* If an obstacle hits the player's lane, one life is lost.
* When all 3 lives are over, GAME OVER is shown.
* Press 'R' to restart the game or any key to exit.
